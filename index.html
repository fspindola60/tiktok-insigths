Busca Vetorial

Semantic TikTok Data Search & Insights

About This Application

Busca Vetorial is a local prototype that uses TikTok public data to generate insights for digital marketing and campaign optimization. It collects metadata (video titles, authors, hashtags, engagement) and vectorizes text content (summaries and comments) for semantic search and analysis.

This project is for educational and research purposes only. It does not store or redistribute TikTok content, comments, or personal user data.
